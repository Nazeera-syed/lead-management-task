# Lead Management Task

## Live URLs
- Frontend (Vercel):https://lead-management-task.vercel.app
- Backend (Render): https://lead-management-backend-imll.onrender.com

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Hosting: Vercel (frontend), Render (backend)

## Features
- Business landing page
- Lead submission form (Name, Email, Phone, Service)
- Admin login 
- Admin dashboard to view all leads
- Filter leads by service type

## API Routes
- POST /api/leads – Create a lead
- GET /api/leads – Fetch all leads
- POST /api/login – Admin login

## Notes
- Uses dummy in-memory data (no database)
